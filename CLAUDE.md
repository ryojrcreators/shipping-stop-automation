# shipping-stop-automation プロジェクトルール

## セキュリティ

- `.env` ファイルは絶対に閲覧・編集・出力しない
- `.env` への変更が必要な場合は、変更内容をユーザーに伝えて手動で対応してもらう
- APIキー・トークン・ログイン情報・スプレッドシートID等の値はコードに直接書かず、必ず環境変数（GitHub Secrets）経由で参照する
- **このリポジトリは Public（公開）**。実行ログも公開されるため、
  お客様の氏名・追跡番号・住所などの個人情報を標準出力（print）に出さない（ログにはマスク表示）。
- 配送会社のメールアドレス・メール文面は、GitHub 側ではなく GAS ミニ・スクリプト側に持たせ、リポジトリには載せない。

## Claude 従量課金対象の利用禁止

- `claude -p` などのヘッドレス/非対話実行、`anthropic` ライブラリの利用、
  GitHub Actions からの Claude CLI 実行は行わない。
- 文字列処理・判定は通常のコード（正規表現等）で行う。
